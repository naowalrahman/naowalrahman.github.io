12th Post:

Hi! It's been 14 days since I wrote on this blog, and a lot has happened since then. In case you're wondering why I haven't been able to write, it's the usual; catching up on homework. Seriously, homework is annoying. Anyways, talking about what's up.

So, the biggest thing is that I ordered the monitor for my new PC yesterday (!!!). Though I was planning on getting either a single 49 inch monitor or two monitors (for a dual monitor setup), my parents said that I was only allowed to have one singular monitor that wasn't as big as the 49-inch monitor that I originally wanted. So, I settled on a single 34-inch monitor from Viotek, which should provide a nice, smooth gaming experience with the Nvidia GeFroce RTX 3070 in my PC, as well as ample space for productivity, though not as much as my previous configurations. It'll be arriving in a week or so, so that's fun!

There's stuff that happened before me ordering this monitor, but it's pretty boring. It was pretty normal stuff - homework, an AP Biology test (which I may have passed, may have failed, depending on how lucky I get). But, the best occasion, which will surprise some and will seem normal to others, happened today!

So, what happened today?

I switched to linux! Well, I didn't completely commit myself to ussing linux, but I have a linux distro on my PC now, so I can dual boot with both linux and Windows 10. The linux distro that I found most appealing and most supported was Ubuntu, so installed Ubuntu 20.04 LTS on my system. I have to admit, though linux is less feature rich that Windows 10, it is certainly a lot faster than Windows 10 and much more programming oriented. Some things that needed lots of setup on Windows 10 just work on Ubuntu. An example of this would be programming with Git and GitHub Desktop. All I had to do to set it up was runsudo apt install git and install a modified version of GitHub Desktop for linux to get my workflow up and running. It even automatically set up source control on Visual Studio Code for me. Programming on this was just such a treat! Obviously, this type of situation can also be applied on Windows 10, where some things on Windows 10 just work, like setting up Google Drive sync. All I had to do on Windows 10 to get Google Drive working was install Google Drive File Stream and log in, and that was it; I had a Google Drive client on my PC. However, with Ubuntu, because the native app wasn't available for linux, I had to try out 3 different third party Google Drive clients until I found something that was actually usable and worked. I tried GoSync, which was impossible for me to setup, then Grive2, which was apparently blocked by Google for "safety reasons," and then I finally tried CloudCross, which worked. I already knew about CloudCross and its efficiency on the enterprise level, but GoSync and Grive2 had an actual GUI and autosync, which is really convenient. However, with CloudCross, not only do I have to manually sync my files, everything is CLI based and local. So, it's not the most convenient setup in the world, but it works.

Other than those types of small inconveniences in Ubuntu/linux in general, I love using it! Its lightweight architecture makes my current computer (not my new one) feel refreshingly fast, and I have a lot more freedom in terms of customization and what I can modify. Using the terminal is great as well, though it can be a pain sometimes when things don't work out. Other than that, I'm enjoying Ubuntu a lot!

Moving on from my experience with linux, lets talk about winter break! It starts this Thursday, and I'm really excited! I won't have much homework according to my teachers, and whatever homework I do have will be done on the first day, so I can sleep 24/7 afterwards (just kidding). Other than grinding homework on the first day, I plan on fully learning the essentials of C++ as well as competitive programming algortihms. Im part of the Stuyvesant Competitive Computing Club, and often times when they teach a new concept, I don't understand any of it. For programming, listening isn't enough. I need to be taught step by step. The club sort of introduces the concept and explains how it works, but I don't get where the solutions are coming from. So, I'm going to relearn more competitive programming skills (things like Big O notation, different algorithms, graphs, etc). Hopefully, I have a productive winter break!\

I have nothing more to say, so thanks for reading! I ranted a little more than usual, so sorry for that. Have a wonderful day, whoever you are.
