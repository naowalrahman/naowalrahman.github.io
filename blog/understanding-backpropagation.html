Understanding Backpropagation

What I learned building a neural network from scratch in C++

Naowal Rahman · November 30, 2024

Recently, I built my own neural network from scratch in C++, and went through the ordeal of learning how gradient descent and backpropagation work. So, I thought sharing my own explanation could help solidify both my understanding and that of any readers. Here goes!

Somewhat unintuitively, backpropagation has 2 steps: forward propagation and backward propagation. We first propagate the input forward through the network to get the activations at each layer. Then, we calculate the error at the output layer and propagate that error "backward" through the layers to calculate gradients for the loss function. I'll explain what all of this means below.

Forward Propagation

Consider all of the inputs X\mathbf{X}X as a n×1n \times 1n×1 matrix, where n is the number of inputs. Then, for each hidden layer from 1…L1 \dots L1…L we have a weight matrix Wl\mathbf{W}^lWl and a bias vector bl\mathbf{b}^lbl. Each value Wijl\mathbf{W}^l_{ij}Wijl​ represents the weight of the connection between the iii-th neuron in layer lll and the jjj-th neuron in layer l−1l - 1l−1. Similarly, each value bil\mathbf{b}^l_ibil​ represents the bias term of the iii-th neuron. All in all, we can represent each activation al\mathbf{a}^lal recursively like this:

zl=Wlal−1+blz^l = \mathbf{W}^l \mathbf{a}^{l-1} + \mathbf{b}^lzl=Wlal−1+bl al=σ(zl)\mathbf{a}^l = \sigma(\mathbf{z}^l)al=σ(zl)

where σ\sigmaσ is the activation function for the layer lll. Note that a0=X\mathbf{a}^0 = \mathbf{X}a0=X.

By starting from the input layer and propagating to layer LLL, we arrive at aL=σ(zl)\mathbf{a}^L = \sigma(\mathbf{z}^l)aL=σ(zl).

Backward Propagation

Now that we've found the output, we should calculate the loss (or cost), measuring the difference between the predicted output with our current weights and biases and the expected output. We measure this with a loss function L\mathcal{L}L:

L(y,y^)=1n∑i=1mL(yi,y^i)\mathcal{L}(\mathbf{y}, \hat{\mathbf{y}}) = \frac{1}{n}\sum_{i=1}^{m}{\mathcal{L}(\mathbf{y}_i,\mathbf{\hat{y}}_i)}L(y,y^​)=n1​i=1∑m​L(yi​,y^​i​)

Now, this is a good time to reinstate our goals: we want to find the gradient of the loss function ∂L/∂Wl\partial \mathcal{L} / \partial \mathcal{\mathbf{W}^l}∂L/∂Wl for each layer lll so that we can nudge the current values in a way that decreases the error. This is because the gradient always points in the direction of greatest change, so by moving in the opposite direction of the gradient of L\mathcal{L}L we slowly minimize it.

The naive way to calculate such a gradient is by actually taking each ∂L/∂Wijl,∀i∈{1,2,…,n},∀j∈{1,2,…,m}\partial \mathcal{L} / \partial \mathcal{\mathbf{W}_{ij}^l}, \forall i \in \{1, 2, \dots, n\}, \forall j \in \{1, 2, \dots, m\}∂L/∂Wijl​,∀i∈{1,2,…,n},∀j∈{1,2,…,m} for a layer with mmm inputs and nnn inputs. However, this is impractical. To do this, we'd need to have training data containing not only input and expected final output but also expected hidden layer outputs. This would make training data neural network specific, meaning one set of training data cannot be used on a slightly altered model. Instead, we can use the multivariate chain rule, which allows us to account for the dependencies between layers:

∂L∂Wl=∂L∂al⋅∂al∂zl⋅∂zl∂Wl\frac{\partial \mathcal{L}}{\partial \mathcal{\mathbf{W}^l}} = \frac{\partial \mathcal{L}}{\partial \mathcal{\mathbf{a}^l}} \cdot \frac{\partial \mathcal{\mathbf{a}^l}}{\partial \mathcal{\mathbf{z}^l}} \cdot \frac{\partial \mathcal{\mathbf{z}^l}}{\partial \mathcal{\mathbf{W}^l}}∂Wl∂L​=∂al∂L​⋅∂zl∂al​⋅∂Wl∂zl​ zl+1=Wl+1al+bl+1\mathbf{z}^{l+1} = \mathbf{W}^{l+1} \mathbf{a}^l + \mathbf{b}^{l+1}zl+1=Wl+1al+bl+1 δl≔∂L∂al⋅∂al∂zl=(∂L∂zl+1⋅∂zl+1∂al)⋅∂al∂zl=(δl+1(Wl+1)T)⋅∂al∂zl\delta^l \coloneqq \frac{\partial \mathcal{L}}{\partial \mathcal{\mathbf{a}^l}} \cdot \frac{\partial \mathcal{\mathbf{a}^l}}{\partial \mathcal{\mathbf{z}^l}} = \left( \frac{\partial \mathcal{L}}{\partial \mathcal{\mathbf{z}^{l+1}}} \cdot \frac{\partial \mathcal{\mathbf{z}^{l+1}}}{\partial \mathcal{\mathbf{a}^l}} \right) \cdot \frac{\partial \mathcal{\mathbf{a}^l}}{\partial \mathcal{\mathbf{z}^l}} = \boxed{ ( \delta^{l+1} \left(\mathbf{W}^{l+1})^T \right) \cdot \frac{\partial \mathcal{\mathbf{a}^l}}{\partial \mathcal{\mathbf{z}^l}} }δl:=∂al∂L​⋅∂zl∂al​=(∂zl+1∂L​⋅∂al∂zl+1​)⋅∂zl∂al​=(δl+1(Wl+1)T)⋅∂zl∂al​​ zl=Wlal−1+bl\mathbf{z}^l = \mathbf{W}^l \mathbf{a}^{l-1} + \mathbf{b}^lzl=Wlal−1+bl ∂L∂Wl=δl⋅∂zl∂Wl=δl(al−1)T\frac{\partial \mathcal{L}}{\partial \mathcal{\mathbf{W}^l}} = \delta^l \cdot \frac{\partial \mathcal{\mathbf{z}^l}}{\partial \mathcal{\mathbf{W}^l}} = \boxed{ \delta^l (\mathbf{a}^{l-1})^T }∂Wl∂L​=δl⋅∂Wl∂zl​=δl(al−1)T​

Thus, we can recursively calculate δl\delta^lδl and hence ∂L/∂Wl\partial \mathcal{L} / \partial \mathcal{\mathbf{W}^l}∂L/∂Wl for each layer. For the neural network described above, this takes O(n2)\mathcal{O}(n^2)O(n2) derivatives for the output layer and depends on the training data only having the predicted result (therefore allowing us to generalize training data to any neural network).

To fully train our neural network, we can simply forward propagate and back propagate on every piece of training data. This method of training a neural network is called stochastic gradient descent.
