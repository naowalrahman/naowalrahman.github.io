KA Project - Animal Attack

Alright, so this is another processing.js Khan Academy project. This was originally created in the Khan Academy environment, but then transferred over to Visual Studio Code. That created problems, though. Since Khan Academy and I use a different version of processing.js (and I don't know which one they use) some features just end up not working when I transfer to VS Code. The problem this time was that I used an arc() command to draw the smile on the dog. It appeared fine on Khan Academy, but completely messed up here once transferred. Basically, the smile went from looking like a smile to just a circle. So, the last 2 parameters of the arc() command weren't read properly, I guess? Anyway, to fix that, I had to use an ellipse() command to cover up the the part of the arc that I didn't want. I tried searching for a more legitimate solution, but I couldn't find any.
