Shooting Star Project

A cool Processing JS animation that may or may not ruin your eyes

Posted on 7/9/2020 at 3:20 PM. 

This was created with HTML and some embedded Processing JS code. The fact that you are able to put a program like this on a web page with a few <script> tags is amazing to me!

For this specific project, I didn't have to change any lines of code when I transferred it over from Khan Academy to this website (and an actual text editor). This means that there are no differences between the version of Processing JS that Khan Academy uses and what I use for just the functions used in this animation. It saved me from the extra work that I had to do with the Animal Attack Project, where some of the code didn't work as intended.

This is by far the best Processing JS animation I've done because it's just so action packed. The RGB background is so energetic (but may also ruin your eyes) and the stars are blinking randomly in a way that makes it seem like something big is happening. Compared to other animations I've created with this library, this just shows me the true capabilities of this JavaScript library. 

Anyways, I just wanted to speak about my thoughts on this Processing JS project. Thanks for reading!
